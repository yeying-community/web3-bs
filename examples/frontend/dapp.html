web3-bs 快速连接钱包 Demo

面向 Web3 前端开发者的标准接入示例：先完成钱包连接，再按需扩展 SIWE 登录与 UCAN 多后端授权。

SDK 地址：/dist/web3-bs.umd.js SDK Loading... 推荐访问：http://127.0.0.1:8001/examples/frontend/dapp.html

标准流程（推荐）

这是集成 `web3-bs` 的最小流程：`getProvider` → `getPreferredAccount` → `watchAccounts/onChainChanged`。

  1. 检测浏览器注入 Provider（YeYing/MetaMask 等）。
  2. 连接并获取优选账户（优先复用上次已连接账户）。
  3. 注册账户和链变更监听，确保 DApp 状态自动同步。
